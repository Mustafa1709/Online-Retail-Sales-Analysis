# 🛒 Online Retail Sales Analysis

## 📌 Project Overview

This project analyzes an online retail dataset using **Python (Pandas)** for data cleaning and exploratory data analysis (EDA), followed by building an interactive **Power BI Dashboard** to visualize business insights.

---

## 🎯 Objectives

- Clean and preprocess raw retail data
- Handle missing values and duplicates
- Standardize country, brand, and weight values
- Perform exploratory data analysis (EDA)
- Create an interactive Power BI dashboard

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- Jupyter Notebook
- Power BI
- Git
- GitHub

---

## 🧹 Data Cleaning

The following cleaning steps were performed:

- Removed duplicate rows
- Handled missing values
- Standardized country names
- Cleaned brand names
- Removed unwanted symbols and emojis
- Standardized weight values
- Converted data types
- Formatted dates
- Saved the cleaned dataset

---

## 📊 Exploratory Data Analysis (EDA)

Performed analysis on:

- Orders by Month
- Orders by Year
- Average Unit Price
- Brand Distribution
- Country Distribution

---

## 📈 Power BI Dashboard

The dashboard contains:

- KPI Cards
- Orders by Month
- Orders by Year
- Orders by Country
- Average Unit Price by Brand
- Interactive Slicers (Country, Month, Brand)

---

## 📌 Key Insights

- Total Orders: **2615**
- Total Brands: **845**
- Total Countries: **56**
- Total Products: **2378**
- Average Unit Price: **12.99**
